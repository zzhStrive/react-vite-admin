## 简介

**react-vite-admin**

基于 React 生态系统搭建的后台管理系统模板

## 技术栈

react@16.10.1 + vite@3.0.7 + antd@3.23.5 + axios@0.19.0

## 快速启动

```
# 克隆项目
git clone git@github.com:zzhStrive/react-vite-admin.git



# 进入项目目录
cd react-vite-admin

# 安装依赖
npm install  / yarn

# 启动服务
npm run dev  / yarn dev

# 打包
npm run build  / yarn build
```
